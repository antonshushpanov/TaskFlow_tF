**TaskFlow** — это современное веб-приложение для управления повседневными делами, проектами и целями. Создано с фокусом на простоту, продуктивность и эстетику. TaskFlow помогает вам перестать забывать важное, сфокусироваться на главном и чувствовать удовлетворение от каждого выполненного пункта.
Основные функции

✅ Создание и редактирование задач

Простой ввод текста
Возможность изменить название задачи в любой момент

🗑️ Удаление и архивирование

Мягкое удаление с возможностью восстановления (опционально)
Полная очистка завершённых задач — одним кликом

🔘 Отметка выполнения

Визуальная индикация прогресса
Автоматическая сортировка: выполненные — внизу

📂 Категории / Списки задач (в разработке)

Разделяйте личные, рабочие и творческие задачи
Переключайтесь между списками без потери контекста

🔍 Поиск и фильтрация

Быстро находите нужную задачу по ключевому слову
Фильтр: «все», «активные», «выполненные»

📅 Дедлайны и напоминания (в планах)

Указывайте сроки выполнения
Настройте уведомления (в будущем — с помощью Notification API)

🎨 Индивидуальный стиль

Переключайтесь между темами

Минималистичный дизайн без отвлекающих элементов

**Технологический стек:**

**Язык:** JavaScript (ES6+) с возможностью последующей миграции на TypeScript, Kotlin.

**Верстка:** HTML5 + CSS3 (с использованием Flexbox и CSS Variables для тем)

# TaskFlow

**TaskFlow** — A smart and minimalist task manager

**TaskFlow** is a modern web application designed to manage your daily tasks, projects, and goals. Built with a focus on simplicity, productivity, and aesthetics, TaskFlow helps you stop forgetting what matters, stay focused on what’s important, and feel satisfaction with every completed task.

## Key Features

✅ **Create and Edit Tasks**  
- Simple text input  
- Ability to rename any task anytime  

🗑️ **Deletion and Archiving**  
- Soft delete with optional restoration  
- One-click complete cleanup of finished tasks  

🔘 **Completion Marking**  
- Visual progress indication  
- Automatic sorting: completed tasks move to the bottom  

📂 **Categories / Task Lists** *(In Development)*  
- Separate personal, work, and creative tasks  
- Switch between lists seamlessly without losing context  

🔍 **Search and Filtering**  
- Quickly locate any task by keyword  
- Filter by: “All,” “Active,” or “Completed”  

📅 **Deadlines and Reminders** *(Planned)*  
- Set task due dates  
- Configure notifications (future integration with the Notification API)  

🎨 **Personalized Styling**  
- Switch between themes  
- Minimalist design free of distracting elements  

## Tech Stack

**Language:** JavaScript (ES6+), with potential future migration to TypeScript or Kotlin  

**Frontend:** HTML5 + CSS3 (utilizing Flexbox and CSS Variables for theming)

# TaskFlow

**TaskFlow** — 스마트하고 미니멀리스트적인 작업 관리자

**TaskFlow**는 일상적인 일정, 프로젝트 및 목표를 관리하기 위한 현대적인 웹 애플리케이션입니다. 단순함, 생산성, 미적 감각에 초점을 두어 개발되었으며, 중요한 일을 잊지 않도록 도와주고 핵심에 집중하게 하며, 하나하나 완료된 작업에서 성취감을 느낄 수 있도록 지원합니다.

## 주요 기능

✅ **작업 생성 및 편집**  
- 간단한 텍스트 입력  
- 언제든지 작업명 변경 가능  

🗑️ **삭제 및 아카이빙**  
- 복구 가능한 소프트 삭제 (선택 사항)  
- 완료된 작업 전체를 한 번의 클릭으로 완전히 정리  

🔘 **완료 표시**  
- 직관적인 진행 상황 시각화  
- 자동 정렬: 완료된 작업은 하단으로 이동  

📂 **카테고리 / 작업 목록** *(개발 중)*  
- 개인용, 업무용, 창작용 작업을 구분  
- 컨텍스트 손실 없이 목록 간 자유롭게 전환  

🔍 **검색 및 필터링**  
- 키워드로 원하는 작업을 빠르게 검색  
- 필터: “전체”, “활성”, “완료됨”  

📅 **마감일 및 알림** *(계획 중)*  
- 작업 마감일 지정 가능  
- 알림 설정 (향후 Notification API를 통해 구현 예정)  

🎨 **개인화된 스타일**  
- 테마 간 전환 가능  
- 산만한 요소 없는 미니멀리스트 디자인  

## 기술 스택

**언어:** JavaScript (ES6+) — 향후 TypeScript 또는 Kotlin으로 마이그레이션 가능성 있음  

**프론트엔드:** HTML5 + CSS3 (Flexbox 및 CSS Variables를 활용한 테마 지원)
